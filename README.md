# todo_app
A To DO web application that has been built using Html(EJS Template), Css, Bootstrap, Node Js, Express Js, MongoDb using MVC architecture.
# Install all Dependencies
## Install Express:
npm install express
## Install Nodemon:
npm install nodemon
## Install Mongoose:
npm install mongoose
## Install EJS Template:
npm install ejs
# To run server using command line:
npm start
#### or 
node index.js
## Note:
Ignore error in home.ejs (if any), it is caused due to usage of ejs code tags.
